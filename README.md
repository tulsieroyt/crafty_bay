# Crafty Bay - Ecommerce Mobile Application

This is an Ecommerce Mobile Application made with flutter framework. 

## Features of the application:
* Splash Screen
* User Authentication
* Shopping cart and checkout
* Add to widhlist
* Review and Rating
* OTP Verification
* Payment Getway

## Tech Stack

**Forntend:** Flutter

**Backend:** RESTful API
